Migrations

Comandos básicos

  • Status => php artisan migrate:status

    Com esse comando conseguimos verificar as situações das Migrations, se foram executadas ou ainda estão com alguma pendência.

  • Reset => php artisan make:reset

    Ele vai reverter todas as migrações do banco, da migração mais atual para a mais antiga, ou seja, faz com que o banco de dados volte para o estado inicial, antes da execução de todas as migrations. É executado o método down

  • Refresh => php artisan make:refresh

    Semelhante ao reset, reverte todas as migrations, executando o método down de cada uma, e na sequência executa o método migration. Reiniciando o banco de dados.

  • Fresh => php artisan make:fresh

    Faz o Drop de todos os objetos do banco de dados + o migrate para recriar os objetos.
